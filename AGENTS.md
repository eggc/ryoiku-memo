# アプリケーション概要

発達障害やその他の障害を持つ子供の療育記録を取るためのアプリです。
父親と母親がメインユーザで、それ以外の利用者は想定していません。
主な機能

- ノート: ほとんどの記録は「ノート」という単位でまとめられる。ノートは一人複数個持てます。
- タイムライン: ノートは複数個のスタンプで構成されたタイムラインを持ちます。
- スタンプ: 寝たとき、起きたときなどのイベントをワンタッチで記録します。1日に複数回利用できます。
- グラフ: スタンプのイベントから睡眠時間をグラフ化してみやすく表示します。
  - 睡眠障害のある子供の記録を分析するときに使いますが、そうでない場合は利用しなくても良いです。
- ログイン:
  - ログインした場合はクラウドストレージ(firestore)に記録を保存できます。
  - ログインしない場合はローカルストレージに記録を保存します。
